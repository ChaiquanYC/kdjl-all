3. PHP 登录页 (login/login.php)

4. PHP vs React 核心 UI 差异

4.1 架构差异

对比项PHP 原版React 新版
页面架构 iframe 多页面: 主框架(index.php) + iframe 动态加载各个 Mod.php SPA 单页应用: React Router + 面板切换，无 iframe
主题色调 棕金色 (#693600 文字, 淡黄/白色背景, 金色边框) 深蓝暗色 (#1a1a2e/#16213e 背景, #eee 文字, 蓝色边框)
视觉风格 图片驱动: 大量 GIF/JPG 背景图、按钮精灵图、装饰边框 CSS 驱动: 纯色背景、CSS 渐变、Emoji 图标、CSS 边框
导航方式 左侧竖排图片按钮 (background-image 切换) 顶部横排文字按钮 (CSS hover 变色)
聊天实现 Flash Socket (snb.php 1262行 Socket服务器) WebSocket STOMP (Java 后端)
游戏区尺寸 固定 788×319px iframe 自适应 flex 布局
背包展示 弹出覆盖层 17格窗口, 带背景装饰图 面板内嵌 grid 布局
