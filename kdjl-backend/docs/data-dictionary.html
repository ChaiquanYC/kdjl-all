1. player (玩家表)

PHP 读取: $_pm['user']->getUserById($uid) → 几乎所有模块。
PHP 写入: FightGate.php (金币/经验), getTask.php (任务状态), Shopsm_Mod.php (购买扣款)
字段类型含义PHP 使用说明
id PK int 玩家唯一ID 所有查询的 uid 条件
name varchar(255) 登录用户名 loginCheck.php 验证; 用作 $_SESSION['username']
password varchar(255) 密码 (实际存储 secret MD5) 注意: 这个字段实际存的是 MD5(password)，不是原文。PHP loginCheck.php 用 secret 字段验证
nickname varchar(255) 游戏内显示昵称 全服公告/聊天/捕捉广播中使用 $_SESSION['nickname']
secret varchar(255) MD5(password) 认证用 loginCheck.php: WHERE secret=MD5($password)
vip int VIP 等级 (0=普通) vip.php 显示; vipbuyGate.php 购买权限检查
money int 金币数量 FightGate.php: 战斗奖励 +money; Shopsm_Mod.php: 购买 -money; 上限 10亿
yb int 元宝 (付费货币) ybbuyGate.php: 元宝购买扣款; pay_Mod.php: 充值增加
score int 积分 活动/排名使用
prestige int 声望值 Prestige_Mod.php: 声望兑换装备; 排行榜使用
active_score int 活跃积分 活动系统使用
mbid int 主战宠物 ID (userbb.id) 核心字段。FightGate.php: 战斗默认使用 mbid 宠物; usedProps.php: 装备穿戴到此宠物; Pets_Mod.php: 设置主战
fightbb int 当前战斗中的宠物 ID FightGate.php: 战斗开始时 mbid → fightbb; 战斗结束后可能恢复
inmap int 当前所在地图 ID Expore_Mod.php: 点击地图 → UPDATE inmap; Fight_Mod.php: 根据 inmap 加载怪物
openmap varchar(255) 已解锁地图列表 (逗号分隔) openMap.php: 使用钥匙道具 → 追加 mapId; "1,2,5" 表示已解锁地图 1/2/5
maxbag int 背包最大容量 get.Catch.php: 掉落时检查 $l >= $user['maxbag']; 默认 30
maxmc int 宠物牧场最大容量 autoCatch.php: $resBBNum >= $resMaxMC 牧场满检查
maxbase int 仓库最大容量 Base_Mod.php 使用
task varchar(255) 当前任务步骤 getTask.php: startTask 设置 task=任务ID; completeTask 清零 task=""
tasklog varchar(255) 任务进度日志 "see:6,giveitem:888:3,killmon:1:2"。completeTask 检查此字段判断条件是否满足
fighttop int PVP 战力排名 King_Mod.php 排行榜
sex varchar(255) 性别 reg1.php: 注册时选择
headimg int 头像ID —
mapinfo varchar(255) 地图扩展信息 —
dblexpflag int 双倍经验标记 (1=开启) FightGate.php: 双倍时间内战斗经验×2
autofitflag int 自动战斗标记 (1=托管中) Tuoguan_Mod.php: 挂机自动战斗开关
sysautosum int 系统自动战斗剩余次数 FightGate.php: 次数用完 → echo 'autoend'
maxautofitsum int 最大自动战斗次数 —
fieldpwd varchar(255) 二级密码 安全验证使用
friendlist varchar(255) 好友列表 (逗号分隔ID) friendGate.php: 添加/删除好友

3. userbb (用户宠物表)

PHP 读取: $_pm['user']->getUserPetById($uid) → Pets_Mod.php, FightGate.php, get.Catch.php
PHP 写入: get.Catch.php (新宠物), Chuancheng.php (传承结果), usedProps.php (装备穿戴)
字段含义PHP 使用说明
id PK宠物唯一ID所有宠物操作的主键
uid所属玩家 ID关联 player.id
name宠物名字 (从 bb.name 复制)战斗显示/捕捉公告/合成界面
level等级FightGate.php: 升级判定; get.Skill.php: 学习技能等级检查
wx五行属性: 1=金 2=木 3=水 4=火 5=土 6=超神 7=神圣FightGate.php: 五行克制计算; Chuancheng.php: wx=6 神宠才能传承; usedProps.php: 装备穿戴五行检查
ac物理攻击力Ack.getSkillAck(): base = (ac + ackvalue) * plus - 对方mc
mc魔法攻击/防御力作为防御方时: 对方的(ac+ackvalue)*plus - 我的mc = 基础伤害
srchp基础 HP (Source HP)最大HP = srchp + addhp。创建时 = bb.hp
addhp附加 HP (装备加成)getzbAttrib(): 装备提供的额外HP; 战斗中加血时优先填充 srchp 再填 addhp
hp当前 HPFightGate.php: $rs['hp'] -= $gwac1; 捕捉时 1 - hp/maxHp
srcmp基础 MP同 srchp 概念
addmp附加 MP (装备)同 addhp 概念
mp当前 MP技能消耗检查 ump > mp → 拒绝
speed速度影响战斗出手顺序
hits命中值Ack.getHitsP(): hitsp = (attack.hits - defend.miss) / 100
miss闪避值同上公式; 钳制 hitsp ∈ [0.1, 1.5]
czl成长率 (全游戏核心指标)getCzl(): 随机生成 如 "1.2~1.5" → rand(1.2, 1.5); 影响宠物战力评价、传承条件 czl≥60
skillist技能列表格式: "skillsysId:level,skillsysId:level"。如 "1:1,302:3" 表示技能1为Lv.1, 技能302为Lv.3
zb装备字符串"pos:bagId,pos:bagId"。如 "0:1283,4:1286" 表示位置0(userbag id=1283)和位置4(userbag id=1286)有装备
kx五行抗性逗号分隔: "金抗,木抗,水抗,火抗,土抗"。战斗时影响伤害计算
nowexp当前经验值FightGate.php: 战斗后 +exps; when nowexp >= lexp → 升级
lexp升级所需经验初始 55-170; 升级后更新
imgstand / imgack / imgdie站立/攻击/死亡 精灵图文件名如 "t353.gif" / "a353.gif" / "d353.gif"; 战斗界面渲染
headimg / cardimg / effectimg头像/卡片/特效图Pets_Mod.php: 宠物卡片显示 cardimg
subyl~subkl7 个成长值 (金/木/水/火/土/生命/速度)从 bb 模板复制; 影响宠物各维度成长
muchang宠物状态1=携带 3=配对中 4=已配对 5=等待确认 6=传承中 7=传承完成; Chuancheng.php/Merge_Mod.php 使用
chchengbb配对/传承对方宠物IDChuancheng.php: 配对时互设
chchengtime传承开始时间24h 倒计时
chchengcz传承次数记录"次数,上次czl" 用于控制传承频率
chchengsx传承时对方的属性快照"ac,mc,srchp,hits,miss,speed,srcmp,level" 用于公式计算
addsx传承累计属性增长记录每次传承的增长量
chchengcolor传承后名字颜色传承完成→ '#FF66CC' 粉色
