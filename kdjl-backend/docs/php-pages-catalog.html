架构总结

模式说明示例
Mod (界面)显示/渲染页面，加载HTML模板文件Fight_Mod.php → tpl_fight.html
Gate (网关)Ajax操作处理器，接收请求→处理逻辑→返回结果FightGate.php → 攻击/捕捉/道具
Kernel (内核)核心业务类，封装MySQL+Memcache数据访问user.v1.php / task.v1.php
Config (配置)游戏常量、道具/技能/任务/怪物数据config.game.php / config.props.php
Template (模板)65个HTML文件，通过file_get_contents+str_replace渲染tpl_fight.html / tpl_smshop.html
