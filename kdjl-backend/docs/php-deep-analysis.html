1. 战斗引擎核心公式

1.1 伤害计算公式 (class Ack, sec_common_fnc.php:2435-2517)

基础伤害:
base = (attacker.ac + ackvalue) × plus - defender.mc
base = max(1, base)

命中修正:
damage = round(base × hitsp) + 1
其中 hitsp = (attacker.hits - defender.miss) / 100，钳制在 [0.1, 1.5]

浮动:
foalt = rand(-10, 5)
final = intval(damage × (100 + foalt) / 100)
即伤害在 -10% 到 +5% 之间浮动

暴击:
if crit: final = final × 2

最低伤害:
final = max(1, final)

  2. 需要升级道具: vary≠4 用道具 733, vary=4 用道具 1666

  4. 检查宠物等级 >= requires[当前等级]
  5. 更新 skill: level+1, value=ackvalue[新等级], plus=plus[新等级]...
  6. 如果 imgeft 是属性加成类 (addac/addmc/addhp/addmp/addhits)，永久修改宠物属性
  7. 消耗升级道具 (userbag sums-1)

7.1 捕捉公式 (get.Catch.php:88-240)

标准捕捉 (props.effect 以 "catch:" 开头):
rate = (catchv / 100) × (1 - currentHP / maxHP) + itemBonus
roll = rand(1, intval(100 / (rate × 100)))
if roll == 1: 捕捉成功!

获取装备 (props.effect 以 "getitems:" 开头):

roll = rand(1, a) where a = rate==0 ? 10000 : intval(100 / (rate×100))
if roll == 1: 随机获取道具 (从 effect 参数解析)

获取特殊物品 (props.effect 以 "get:" 开头):
rate = (catchv / 100) × (1 - currentHP / maxHP) + itemBonus
if roll == 1: 获取指定道具 + 可选公告

8. 宠物创建与成长率

8.1 成长率系统 (czl)

宠物的核心战力指标，由字符串表示如 "1.2~1.5"，getCzl() 函数在此范围内随机：

// 解析 czl 字符串 (如 "1.2~1.5")
function getCzl($czl_str) {
    list($min, $max) = explode('~', $czl_str);
    $czl = rand($min * 100, $max * 100) / 100;
    return $czl;
}

12. 合成 & 结婚系统 (Merge/merge)

12.1 婚姻系统概述

PHP 游戏中的宠物合成系统实际上是一个玩家间结婚系统。两名玩家通过求婚→接受→结婚的流程建立婚姻关系，然后可以进行宠物培育。

12.2 婚姻状态机 (player_ext 表)

字段值含义
merge0 / player_id配偶 ID (0=未婚)
request_merge0 / player_id向谁求婚 (或 0=无)
request0/1/2/30=正常, 1=请求离婚, 2=已拒绝, 3=拒绝不通知
send"count,prop_id"求婚时的定情信物
nomergetimetimestamp提出离婚的时间
sjint水晶数量 (离婚消耗)

12.3 结婚流程 (merge.php type=4→3)

  1. 赠送定情信物 (type=4): 玩家A选择背包中的道具 (merge=1标记的求婚道具) 赠送给玩家B
    → UPDATE player_ext SET request_merge=B_id, send='count,prop_id' WHERE uid=A_id
    → 对方收到通知 "玩家A向你提出求婚"
  2. 接受婚姻 (type=3): 玩家B接受玩家A的求婚
    → 将求婚道具发放给B的背包 (INSERT/UPDATE userbag)
    → UPDATE both player_ext: merge=B, request_merge=0, merge=A, request_merge=0
    → 系统广播: "恭喜【A】和【B】结为夫妻！"

13.4 核心传承公式 (type=8 取回)

传承珠倍率: 从道具 effect 字段读取，格式 "chuanc:X"，其中 X = 传承珠的倍率系数

新属性计算公式:
new_stat = intval( ( base_stat + intval(own_stat × own_level / 400) + intval(mate_stat × mate_level / 800) ) × pearl_multiplier )

各属性:
new_ac = intval((rs.ac + intval(own.ac × own.level / 400) + intval(mate.ac × mate.level / 800)) × pearl)
new_mc = intval((rs.mc + intval(own.mc × own.level / 400) + intval(mate.mc × mate.level / 800)) × pearl)
new_srchp = intval((rs.hp + intval(own.srchp × own.level / 400) + intval(mate.srchp × mate.level / 800)) × pearl)
new_hits = intval((rs.hits + intval(own.hits × own.level / 400) + intval(mate.hits × mate.level / 800)) × pearl)
new_miss = intval((rs.miss + intval(own.miss × own.level / 400) + intval(mate.miss × mate.level / 800)) × pearl)
new_speed = intval((rs.speed + intval(own.speed × own.level / 400) + intval(mate.speed × mate.level / 800)) × pearl)
new_srcmp = intval((rs.mp + intval(own.srcmp × own.level / 400) + intval(mate.srcmp × mate.level / 800)) × pearl)

其中:
rs = 宠物种族原始模板数据 (从 bb.pets 缓存读取)
own = 自己的宠物当前属性 (chchengsx 字段存储的对方属性)
mate = 对方宠物存入的属性
pearl = 传承珠倍率 (来自道具 effect 字段，如 "chuanc:1.5")
own.level / 400 = 等级贡献系数
mate.level / 800 = 对方等级贡献系数 (只有自己的一半)
